# 滑板招式成就系统（SkateGoal）产品设计方案 (v1.0 - 极简工具版)

## 一、 产品定位
一款面向滑板爱好者的“打怪升级”记录工具。**核心聚焦于个人成长记录与招式管理，剥离复杂社交，回归纯粹的成就感体验。**

## 二、 核心功能模块

### 1. 用户模块 (Profile)
*   **注册与登录：** 微信一键授权登录（获取头像、昵称）。
*   **滑手名片：** 包含用户专属 ID、站位（Goofy 右脚前 / Regular 左脚前）、滑板板龄、常驻城市。
*   **个人成就：** 仅展示当前点亮的动作数量和获得的头衔。

### 2. 成就与技能树系统 (Achievements & Titles)
*   **动作成就（Trick 粒度）：**
    *   **核心痛点解决：** 针对滑板招式组合多（4种脚位 x 旋转方向 x 翻板方式）导致列表冗长的问题，采用 **“根招式聚合 (Trick Family)”** 逻辑。
    *   **交互逻辑：** 列表中仅展示“根招式”（如 Kickflip）。点击后展开**“脚位矩阵面板”**，分别管理 Normal / Fakie / Switch / Nollie 四种形态的进度。
    *   **每一个动作组合拥有三种状态（更懂滑板人的黑话）：**
        1.  **🎫 体验卡 (Trial)：** 刚开始练，偶尔成一个，运气好能落，但不稳。
        2.  **🔥 死磕中 (Grinding)：** 重点攻克的项目。成招率低，正在反复练习肌肉记忆。
        3.  **🦶 一脚一个 (Mastered)：** 肌肉记忆已固化，随叫随到，风格稳健。
*   **技能树探索模式（Skill Tree View）：**
    *   提供可视化进阶指引（如 Ollie -> BS 180）。
*   **进阶头衔系统（Titles/Badges）：**
    *   解锁特定动作组合后自动获得，仅用于个人主页展示。

### 3. 成长时光轴与汇总页 (Timeline & Summary)
*   **数据大盘：** 直观展示已解锁动作总数、坚持天数。
*   **滑板历程（时光轴）：** 自动生成“X年X月X日，解锁 [动作名]”的日志。

## 三、 第一版 (MVP) 极简页面架构与 UI 草图

核心思路是**“纯工具，无社交干扰”**。底部导航栏（Tab Bar）精简为 2 个：

### Tab 1: 招式库 (核心录入页)
**设计思路**：**“根招式聚合 (Root Trick)” + “脚位快捷面板”**
*   **列表层**：只展示如 `Ollie`, `Shuvit`, `Kickflip`, `Heelflip` 等基础“根动作”，极大缩短列表长度。
*   **交互层**：点击任意根动作，底部弹起半屏面板，展示该动作的 4 种脚位（Normal, Fakie, Switch, Nollie）进度。

```text
┌─────────────────────────┐
│       动作字典 (首页)   │
├─────────────────────────┤
│ [🔍]  [基础] [进阶] [弧面]│
├─────────────────────────┤
│ ▼ 平地动作 (Flatground) │
│                         │
│ [图标] Ollie            │
│  └─ Norm:🦶 Fak:🎫 Sw:🔥│
│                         │
│ [图标] Kickflip         │
│  └─ Norm:🎫 Fak:-- Sw:--│
│                         │
│ [图标] Treflip          │
│  └─ (点击展开脚位面板)   │
│                         │
├─────────────────────────┤
│     [🎯动作库] [👤我的]   │
└─────────────────────────┘
```
*点击 `Kickflip` 后弹出的底部面板（Action Sheet）：*
```text
┌─────────────────────────┐
│      Kickflip 状态管理   │
├──────┬──────┬──────┬────┤
│Normal│ Fakie│Switch│Nol │
│  🦶  │  🎫  │  🔥  │ -- │
├──────┴──────┴──────┴────┤
│ 状态说明：               │
│ 🎫 体验卡 (偶尔成)       │
│ 🔥 死磕中 (重点练)       │
│ 🦶 一脚一个 (稳如狗)     │
└─────────────────────────┘
```

### Tab 2: 我的 (核心展示页)
**目标**：自我满足，截图分享。

```text
┌─────────────────────────┐
│     我的滑板档案        │
├─────────────────────────┤
│ (头像) @阿强  ID: 10024 │
│ 🏆 佩戴头衔：[翻板大王] │
│ 🛹 修练进度：12 招      │
├─────────────────────────┤
│ 📅 成就时光轴：         │
│ ├ 2024.03.15 🦶 Treflip │
│ ├ 2024.01.02 🎫 Kickflip│
│ └ 2023.10.11 🦶 Ollie   │
├─────────────────────────┤
│     [🎯动作库] [👤我的]   │
└─────────────────────────┘
```
*交互说明：*
- **时光轴**：清晰记录每一个里程碑。
- **状态图标**：
  - 🦶 **一脚一个**：代表该动作已完全掌握。
  - 🎫 **体验卡**：代表实现了0到1的突破（即使还不稳）。
  - 🔥 **死磕中**：代表最近正在重点练习（不计入“已成招”统计，但显示在列表中）。

## 四、 后期演进与可扩展方向（社交/图文/视频）

**注：MVP 阶段暂不开发，待工具价值验证后再迭代。**

1.  **社交化 (Social)：** 引入“搜人”、“关注”、“Props (点赞)”机制。
2.  **凭证系统 (Showcase)：** 支持为已解锁动作上传图文/外链/视频凭证，构建“高光时刻”信息流。
3.  **线下结合 (Spots)：** 记录动作解锁的地点。
4.  **排行榜 (Leaderboard)：** 基于地域或动作数量的轻量级排行。
